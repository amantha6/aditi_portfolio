-   **Colors**: Open `style.css` and modify the variables in the `:root` section (top of the file) to change the color scheme.
-   **Images**: Add your images to an `assets` folder (you'll need to create one) and update the `src` attributes in `index.html`. Currently, placeholders are used.
-   **Icons**: We use FontAwesome 6.4.0 (CDN linked in `index.html`).

## ğŸ“¦ Hosting on GitHub Pages

1.  Create a new repository on GitHub (e.g., `my-portfolio`).
2.  Upload these files (`index.html`, `style.css`, `script.js`, `README.md`) to the repository.
3.  Go to **Settings** > **Pages**.
4.  Under **Source**, select `main` (or `master`) branch.
5.  Click **Save**.
6.  Your site will be live at `https://<your-username>.github.io/my-portfolio/`.
